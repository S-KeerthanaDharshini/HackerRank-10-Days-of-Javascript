
# [Day 2: Conditional Statements: Switch](https://www.hackerrank.com/challenges/js10-switch/problem?isFullScreen=true)

## Problem Description

Complete the `getLetter(s)` function that takes a string `s` consisting of lowercase English alphabetic letters (a-z) and returns `A`, `B`, `C`, or `D` based on the following criteria:

- If the first character of `s` is in the set `{a, e, i, o, u}`, return `A`.
- If the first character of `s` is in the set `{b, c, d, f, g}`, return `B`.
- If the first character of `s` is in the set `{h, j, k, l, m}`, return `C`.
- If the first character of `s` is in the set `{n, p, q, r, s, t, v, w, x, y, z}`, return `D`.


## Sample Input

```
adfgt
```

## Sample Output

```
A
```
